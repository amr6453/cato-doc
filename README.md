# 📑 CatoDoc | كاتو دوك - نظام إدارة الحجوزات الطبية الحديث

**CatoDoc** هو منصة رقمية متطورة مصممة لتسهيل عملية حجز المواعيد الطبية وتسهيل التواصل بين الأطباء والمرضى. يجمع النظام بين القوة البرمجية في الخلفية (Backend) والواجهة العصرية السلسة في الأمام (Frontend) لتقديم تجربة مستخدم استثنائية.

---

## 🚀 الميزات الرئيسية

### 🧑‍⚕️ للأطباء:
- **لوحة تحكم متكاملة:** إدارة المواعيد، تتبع الإحصائيات، ومعرفة الدخل الإجمالي.
- **إدارة التوافر:** تحديد مواعيد العمل والـ Slots المتاحة بسهولة.
- **تنبيهات فورية:** استلام إشعارات حية (Real-time) عند حجز أي موعد جديد عبر WebSockets.

### 🤒 للمرضى:
- **بحث ذكي:** تصفية الأطباء حسب التخصص، التقييم، والسعر.
- **حجز مرن:** اختيار الموعد المناسب من بين الأوقات المتاحة للطبيب.
- **لوحة تحكم المريض:** متابعة حالة الحجوزات (قيد الانتظار، مؤكدة، أو مكتملة).

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

### 🔙 الجانب الخلفي (Backend):
- **Django & DRF:** لبناء الـ APIs القوية والمؤمنة.
- **Django Channels & Redis:** للتعامل مع الإشعارات الفورية (Real-time Notifications).
- **PostgreSQL:** لقاعدة بيانات موثوقة (يمكن تغييرها حسب البيئة).
- **JWT Auth:** لتأمين الدخول وحماية بيانات المستخدمين.

### 🎨 الجانب الأمامي (Frontend):
- **Next.js 14 (App Router):** لسرعة الأداء وتحسين محركات البحث (SEO).
- **Tailwind CSS & Shadcn UI:** لتصميم واجهات عصرية ومتجاوبة (Responsive).
- **Lucide React:** لمجموعة الأيقونات العصرية.

## ⚙️ البدء بالعمل

### 🐍 تشغيل الباكيند (Backend):
1. قم بإنشاء بيئة افتراضية: `python -m venv venv`
2. تفعيل البيئة: `venv\Scripts\activate` (على ويندوز)
3. تثبيت المكتبات: `pip install -r requirements.txt`
4. تشغيل السيرفر: `python manage.py runserver`

### ⚛️ تشغيل الفرونت إند (Frontend):
1. الدخول لمجلد المشروع: `cd frontend`
2. تثبيت المكتبات: `npm install`
3. تشغيل وضع التطوير: `npm run dev`
